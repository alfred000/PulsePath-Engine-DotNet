# 🚀 PulsePath Engine - Implémentation .NET

Ce dépôt contient la version C# du moteur PulsePath. L'objectif est d'implémenter les règles métier de vélocité santé définies dans mon étude de cas BA.

📘 **Note d'Architecture** : Ce code a été écrit en appliquant la méthodologie TDD à partir des Spécifications Fonctionnelles Détaillées rédigées dans mon [Étude de cas Business Analyst principal](https://github.com/alfred000/PulsePath-Engine-BA-Case-Study).

## 🛠 Tech Stack
- Langage : C#
- Framework : .NET 8 (Console App)
- Architecture : Orientée Objet (POO)
